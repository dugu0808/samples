# javacsvDemo

使用外部包java处理CSV文件。

#### 依赖

1. 方法一：直接下载javacsv的Jar包：[javacsv2.0](http://xiazai.jb51.net/201807/yuanma/javacsv2.0(jb51.net).rar)
2. 方法二：利用Maven导入：

```xml
<dependency>
  <groupId>net.sourceforge.javacsv</groupId>
  <artifactId>javacsv</artifactId>
  <version>2.0</version>
</dependency>
```
